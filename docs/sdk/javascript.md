# JavaScript SDK

The official JavaScript/TypeScript SDK for StremThru.

[![NPM Version](https://img.shields.io/npm/v/stremthru?style=flat-square)](https://npmjs.com/package/stremthru)

## Installation

```sh
# pnpm
pnpm add stremthru

# npm
npm install stremthru

# yarn
yarn add stremthru
```

## Usage

```ts
import { StremThru } from "stremthru";

const st = new StremThru({
  baseUrl: "http://127.0.0.1:8080",
  auth: "user:pass",
});
```
