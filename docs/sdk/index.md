# SDKs

StremThru provides official SDKs for various programming languages.

## Available SDKs

| SDK                        | Package                                                | Install                 |
| -------------------------- | ------------------------------------------------------ | ----------------------- |
| [JavaScript](./javascript) | [`stremthru`](https://www.npmjs.com/package/stremthru) | `npm install stremthru` |
| [Python](./python)         | [`stremthru`](https://pypi.org/project/stremthru/)     | `pip install stremthru` |

Both SDKs provide a client for the StremThru API with typed interfaces for all endpoints.
