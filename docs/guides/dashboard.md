# Dashboard

The Dashboard is StremThru's admin interface for monitoring and managing your instance. It's available at `/dash/` and requires admin authentication.

::: info
Only users listed in [`STREMTHRU_AUTH_ADMIN`](/configuration/#authentication) can access the dashboard.
:::

## Accessing the Dashboard

Navigate to `{BASE_URL}/dash/` and sign in with your admin credentials. Sessions are cookie-based and persist for 7 days.

## Stats

The main dashboard page shows an overview of your instance:

- **Server Uptime** — how long the instance has been running
- **Torrent Cache Stats** — cache hit/miss rates
- **IMDB Titles** — count of indexed IMDB titles
- **List Stats** — statistics for imported lists (AniList, Letterboxd, MDBList, TMDB, Trakt, TVDB)

## Workers

Monitor background workers that handle tasks like indexer syncing, list imports, and cache maintenance.

- View job logs with success/failure status
- Clean up temporary files generated by workers

## Proxy

Generate proxified links with optional settings:

- **Encryption** — encrypt the proxied URL
- **Expiration** — set an expiry time for the link

## Vault

::: info
Vault features are only available when [`vault`](/configuration/features) feature is enabled.
:::

Vault provides encrypted storage for credentials and account management.

### Stremio Accounts

Add and manage Stremio accounts. You can:

- Add/edit/delete Stremio accounts
- View linked userdata
- Sync account data

### Trakt Accounts

Manage Trakt OAuth connections:

- Add Trakt accounts
- Authenticate via OAuth flow
- View connection status

### Torznab Indexers

Configure torrent indexer sources (e.g. Jackett):

- Add indexers with URL and API key
- Configure rate limits per indexer
- Test indexer connectivity

### Newznab Indexers

Configure Usenet indexer sources:

- Add indexers with URL and API key
- Configure rate limits per indexer
- Test indexer connectivity

## Usenet

::: info
Usenet features require [Vault](/configuration/features) to be enabled. See the [Usenet Setup](/guides/usenet-setup) guide for initial configuration.
:::

### Config

View current Usenet configuration including cache sizes, TTL, and stream buffer settings.

### Servers

Manage NNTP server connections — add, edit, test, and remove Usenet providers.

### Indexers

Manage Newznab indexers — same as the Vault Newznab Indexers section.

### NZB Management

Upload, inspect, and manage NZB files:

- **NZB List** — view all managed NZB files
- **NZB Inspector** — inspect NZB file contents (segments, files, groups)

### NZB Queue

Monitor the download queue with status tracking for queued NZB downloads.

### Pool

View and rebuild the Usenet connection pool — useful for diagnosing connectivity issues.

## Sync

::: info
Sync features require [Vault](/configuration/features) to be enabled.
:::

### Stremio to Stremio

Create sync links between Stremio accounts to keep addon configurations and library items in sync.

### Stremio to Trakt

::: info
Requires [Trakt integration](/integrations/trakt) to be configured.
:::

Sync Stremio watchlist and library with Trakt.

## Settings

### Rate Limit Configs

Create and manage rate limiting rules with configurable time windows and request limits. These can be applied to indexers.
