# Store Addon

The Store addon lets you browse and search content in your debrid store catalog directly from Stremio.

**Path:** `/stremio/store`

## Overview

- Browse your store catalog within Stremio
- Search across stored content
- Multi-store support
- Usenet and WebDL content support

## Configuration

| Environment Variable                         | Description                    |
| -------------------------------------------- | ------------------------------ |
| `STREMTHRU_STREMIO_STORE_CATALOG_ITEM_LIMIT` | Max items to fetch for catalog |
| `STREMTHRU_STREMIO_STORE_CATALOG_CACHE_TIME` | Cache time for catalog         |
