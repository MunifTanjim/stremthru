# Stremio Addons

StremThru includes six built-in Stremio addons that enhance your streaming experience.

## Available Addons

| Addon                  | Path                | Description                                |
| ---------------------- | ------------------- | ------------------------------------------ |
| [Store](./store)       | `/stremio/store`    | Browse and search your store catalog       |
| [Wrap](./wrap)         | `/stremio/wrap`     | Wrap other Stremio addons with StremThru   |
| [Sidekick](./sidekick) | `/stremio/sidekick` | Extra features for Stremio                 |
| [Torz](./torz)         | `/stremio/torz`     | Torrent indexer integration                |
| [Newz](./newz)         | `/stremio/newz`     | Stream Usenet content via Newznab indexers |
| [List](./list)         | `/stremio/list`     | Generate catalogs from external lists      |

## Installing Addons

Each addon has a configuration page accessible from your StremThru instance. Navigate to the addon's path (e.g., `http://your-stremthru:8080/stremio/store`) to configure and install it in Stremio.
