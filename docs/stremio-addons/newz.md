# StremThru Newz

The Newz addon integrates Usenet with Stremio.

**Path:** `/stremio/newz`

## Features

- Indexers:
  - Generic Newznab
  - StremThru (aggregator)
  - Torbox
- NNTP Streaming
- Debrid support
