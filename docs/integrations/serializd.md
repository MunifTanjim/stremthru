# Serializd Integration

[Serializd](https://www.serializd.com/) integration enables TV show list support for Stremio catalogs.

## Used For

- Serializd TV show lists as Stremio catalogs via the [List addon](/stremio-addons/list)

## Setup

No authentication is required. Serializd integration requires [TMDB](/integrations/tmdb) to be enabled.

Check [documentation](/configuration/integrations#serializd).
